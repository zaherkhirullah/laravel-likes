### INSTALLATION STEPS

<i> Step 1 </i>

```
$ composer require hayrullah/laravel-visits
 ```

<i> Step 2 </i>

```
$ php artisan vendor:publish --provider="Hayrullah\Visits\VisitServiceProvider" 
```
